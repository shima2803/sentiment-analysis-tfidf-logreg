# sentiment-analysis-tfidf-logreg
Classificador de sentimentos (positivo/negativo) em reviews de produtos usando TF-IDF + Regressão Logística. Pipeline completo em scikit-learn com limpeza de texto, otimização de hiperparâmetros via GridSearchCV, validação cruzada, avaliação por matriz de confusão e modelo serializado com joblib pronto para deploy.
